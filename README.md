## Overview
This project analyzes sales performance to understand revenue trends, top products, and customer behavior.

## Objectives
- Analyze total sales and profit
- Identify top-performing Category
- Track monthly sales trends

## Tools Used
- Microsoft Excel
- Power BI

## Key Insights
- Sales peaked during high-demand months
- A small number of products generated most revenue
- Customer purchasing behavior shows seasonal patterns
